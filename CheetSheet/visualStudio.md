# Visual Studio Cheet Sheet

## ビルド

| Cmd       | 説明       |
| --------- | ---------- |
| CTL+SFT+B | ビルド     |
| CTL+Break | キャンセル |
| CTL+F7    | コンパイル |
|           |            |

---

## デバック

| Cmd        | 説明                        |
| ---------- | --------------------------- |
| CTL+F5     | デバックなしで開始          |
| SFT+F5     | デバック停止                |
| CTL+SFT+F9 | すべてのブレークPを削除する |
| CTL+F7     | コンパイル                  |
| CTL+F10    | カーソル行まで実行 |


## 編集

| Cmd        | 説明               |
| ---------- | ------------------ |
| CTL+F12    | 宣言に移動         |
| F12        | 定義へ移動         |
| CTL+Space  | 候補表示           |
| CTL+W      | 単語単位で選択     |
| CTL+Delete | 単語単位で削除     |
| CTL＋M,M   | アウトラインの開閉 |
| CTL＋F     | 検索               |
| CTL+H      | 置換               |
|            |                    |

## プロジェクト

| Cmd       | 説明             |
| --------- | ---------------- |
| CTL+SFT+A | 新しい項目の追加 |
| CTL+ALT+A | 既存の項目の追加 |
|           |                  |
